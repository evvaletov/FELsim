# Bibliography

## UH FEL

- arXiv:2510.14061v1 — S. Weinberg, L. Fisher, S. Li,
  "FEL oscillator simulation for the UH MkV FEL."
  Undulator Twiss targets (Table I), oscillator parameters.

- M. Hadmack, Rev. Sci. Instrum. **84**, 063302 (2013) —
  Energy spread measurement for the UH MkV FEL.

- UH-044, M. Hadmack — MkIII dipole modelling (pp. 48–81).
  Measured Enge coefficients for FC1/FC2 chicane dipoles.

## Accelerator Physics

- H. Wiedemann, *Particle Accelerator Physics*, 4th ed. (Springer, 2015).
  Alpha-magnet (p. 449), linac theory (p. 628).

## Simulation Codes

- S. Reiche, "GENESIS 1.3: a fully 3D time-dependent FEL simulation code,"
  Nucl. Instrum. Methods A **429**, 243–248 (1999).
  Genesis4 is the successor code.

- A. Latina, "RF-Track Reference Manual."
  Available in `manuals/RF_Track_reference_manual.pdf`.

- K. Makino and M. Berz, "COSY INFINITY Version 9,"
  Nucl. Instrum. Methods A **558**, 346–350 (2006).

## Standards

- [PALS — Particle Accelerator Lattice Standard](https://pals-project.readthedocs.io/).
  FELsim v2 lattice format follows PALS naming conventions.
